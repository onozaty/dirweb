# dirweb

dirweb is a simple web server.  
You can publish the directory as a web server.

## Usage

```
$ dirweb -d doc -p 8888
```

The arguments are as follows.

```
Usage of dirweb:
  -d, --dir string   dir (default "./")
  -h, --help         help
  -p, --port int     port (default 3000)
```

## Install
